# Contributing

Thanks for contributing to Postman Parallel Collection Runner.

## Development Setup
1. Install Node.js 14+
2. Install dependencies: `npm install`
3. Run tests: `npm test`

## Pull Request Guidelines
- Keep changes focused and minimal.
- Add or update docs for behavior/config changes.
- Include clear reproduction steps for bug fixes.
- Ensure tests pass before opening a PR.

## Issue Guidelines
Use the issue template and include:
- environment details
- command used
- sanitized config
- verbose logs

## Code Style
- Follow existing project style.
- Prefer clear naming and small functions.
